JustGiving API PHP SDK - Reference Implementation

This is a reference PHP implementation for the JustGiving API.
Requires PHP 5.1 or above, php_curl and json support.

A quick example - Retrieving page data

$client = new JustGivingClient("http://api.local.justgiving.com/", "your-api-key", 1, "apiunittests@justgiving.com", "password");
$page = $client->Page->Retrieve("rasha25");
echo "pageShortName - " . $page->pageShortName;
echo "eventName - " . $page->eventName;
