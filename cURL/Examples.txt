CURL Examples

using the Username apitests@justgiving.com and the password "password".

For methods that require authentication, you must add --header "Authorization: Basic {Base64 encoded string of username:password}"
for methods where authentication is optional you can leave the header out, and you will recieve the unauthenticated view of the data. 
Some values may be ommitted.

List Pages
	curl https://api.justgiving.com/YOUR-KEY-HERE/v1/fundraising/pages --header "Authorization: Basic YXBpdGVzdHNAanVzdGdpdmluZy5jb206cGFzc3dvcmQ="
	
Get Page
	curl https://api.justgiving.com/YOUR-KEY-HERE/v1/fundraising/pages/greatswims --header "Authorization: Basic YXBpdGVzdHNAanVzdGdpdmluZy5jb206cGFzc3dvcmQ="
	
Get Page Donations
	curl https://api.justgiving.com/YOUR-KEY-HERE/v1/fundraising/pages/greatswims/donations --header "Authorization: Basic YXBpdGVzdHNAanVzdGdpdmluZy5jb206cGFzc3dvcmQ="
	
Get Pages for registered user
	curl https://api.justgiving.com/YOUR-KEY-HERE/v1/account/david@justgiving.com/pages
